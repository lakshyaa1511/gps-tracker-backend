# GPS Tracker Platform

This project is a GPS tracking platform built using Traccar as the telematics backend.
It provides APIs and services for live tracking, device history, and fleet monitoring.

## Tech Stack
- Backend: Python
- Traccar Server
- REST APIs
- PostgreSQL (if applicable)

## Features
- Live device tracking
- Route and history playback
- Traccar API integration
- Fleet and device management

frontend/  -> Frontend (if applicable)
